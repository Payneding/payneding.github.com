MarkDown Basic

标题设置

在文字开头加上 “#”，通过“#”数量表示几级标题。（一共只有1~6级标题，1级标题字体最大）

代码： #这是一级标题

效果：

这是一级标题

代码： ##这是二级标题

效果：

这是二级标题

代码： ###这是三级标题

效果：

这是三级标题

列表

HTML 列表分无序列表 (unordered list, ul) 和有序列表 (ordered list, ol) 两种。在 Markdown 中用（星号、加号、减号)开始一行表示无序列表，用(数字)开始一行表示有序列表。

无序列表

代码：

- 第一行

- 第二行

- 第三行

效果:

  • 第一行

  • 第二行

  • 第三行

有序列表

1.这是有序列表

2.这是有序列表

3.这是有序列表

嵌套列表

代码：

- 外层嵌套列表

+ 嵌套1

+ 嵌套2

+ 嵌套3

- 外侧嵌套列表

效果：

  • 外层嵌套列表

  • 嵌套1

  • 嵌套2

  • 嵌套3

  • 外侧嵌套列表

横线

三个或以上星号、减号或者下划线单独放在一行即可生成一条横线。

代码：

***

效果：

粗体和斜体

用星号或下划线来实现。用两个 * 包含一段文本就是粗体的语法，用一个 * 包含一段文本就是斜体的语法

*这是斜体语法*

**这也粗体语法**

这是斜体语法

这也粗体语法

插入图片

插入图片的语法。

![alt text](/path/to/img.jpg "Title")

插入连接

行内链接语法：

代码： [this is a link](http://www.baidu.com)

效果：this is a link

代码

在一般的段落文字中，你可以使用反引号 ` 来标记代码区段。

如果要建立一个已经格式化好的代码区块，只要每行都缩进 4 个空格。例如：

python:

dict1={'name':jack,'age':20,'job':'IT'}
for key,value in dict1.iteritems():
    print key,value

仅做测试

ddddff

反斜杠(转义符）

Markdown 可以利用反斜杠来插入一些在语法中有其它意义的符号，例如：如果你想要用星号加在文字旁边的方式来做出强调效果（但不用 标签），你可以在星号的前面加上反斜杠：

Markdown 支持以下这些符号前面加上反斜杠来帮助插入普通的符号：

\ 反斜线

` 反引号

* 星号

_ 底线

{} 花括号

[] 方括号

() 括弧

# 井字号

+ 加号

- 减号

. 英文句点

! 惊叹号
